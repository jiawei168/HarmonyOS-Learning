## 🛠️ 开发环境

### 推荐工具
- **DorfGo Studio**: 主要开发IDE
- **OpenHarmony SDK**: 开发套件
- **模拟器**: 应用测试环境

### 工程结构说明
- `app.json5`: 应用配置文件
- `module.json5`: 模块配置文件
- 其他配置文件：workspace.xml, modules.xml等
